# Personajes Smash — Galería con Bootstrap

Página que presenta **personajes** inspirados en el universo tipo *Super Smash Bros.*: rejilla responsive, estilos personalizados y **Bootstrap** para componentes y rejilla.

**Autora del repositorio:** [@Juli21v](https://github.com/Juli21v)

## Stack

- HTML, CSS, JavaScript
- Bootstrap (archivos en `boostrapdownloads/`)

## Estructura del repositorio

```
Personajes-smash/
├── index.html
├── css/style.css
├── js/script.js
└── boostrapdownloads/    # CSS y JS de Bootstrap locales
```

## Cómo verlo

Abre `index.html` en el navegador.

## Nota

El nombre de la carpeta `boostrapdownloads` conserva la grafía del proyecto original.
